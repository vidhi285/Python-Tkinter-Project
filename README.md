# Python Tkinter Project

A collection of GUI applications and games developed using Python, Tkinter, and Pygame.

---

## Overview

This project contains multiple desktop-based GUI applications and mini games created using Python. The project demonstrates graphical user interface development, event handling, game logic, multimedia integration, and user interaction using Tkinter and Pygame.

The applications included in this project are:

* Snake Game
* Pong Game
* Tic Tac Toe
* Memory Puzzle
* Driving License Exam Simulator
* Music Player
* Multiple Question GUI Programs
* Purchase Movie-Ticket
* Driving Lisence Exam

---

## Technologies Used

### Python

Python is a high-level, interpreted programming language known for its simplicity and readability. It is widely used for software development, automation, data science, artificial intelligence, and GUI applications.

### Tkinter

Tkinter is Python’s standard GUI (Graphical User Interface) library. It is used to create desktop applications with windows, buttons, labels, menus, text boxes, and event-driven functionalities.

### Pygame

Pygame is a Python library designed for developing multimedia applications and games. It provides support for graphics, sound, animations, and keyboard/mouse interactions.
